# CodifyAI

An AI-powered full-stack application builder that transforms natural language prompts into modern web applications. Built with Spring Boot 4.0.1 and Java 17, CodifyAI provides a scalable backend for AI-driven code generation, project management, authentication, real-time interactions, and seamless integration with modern frontend frameworks and AI models.
